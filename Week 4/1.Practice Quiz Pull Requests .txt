Q 1) What is the difference between using squash and fixup when rebasing?
Ans : Squash combines the commit messages into one. Fixup discards the new commit message

Q 2) What is a pull request?
Ans : A request send to owner and collaborators of the target repository to pull your recent changes

Q 3) Under what circumstances is a new fork created?
Ans : When you want to experiment with changes without affecting the main repository .

Q 4) What combination of command and flags will force Git to push the current snapshot to the repo as it is, possibly resulting in permanent data loss?
Ans : git push -f
